Etiquetas HTML

  • DOCTYPE:

    Define el tipo de documento con el que estamos trabajando. Comando ! + enter.

  • HTML:

    Define el tipo de documento.

  • HEAD:

    Información básica del documento (metadatos, hojas de estilo para css, etc.).

  • HEADER:

    Define la cabecera del proyecto. Usualmente contiene el nombre del sitio, además de la barra de navegación principal.

  • BODY:

    Representa el contenido básico del proyecto.

  • MAIN:

    Contiene la información más relevante e importante de todo el proyecto. SOLO PUEDE HABER UN MAIN POR TRABAJO.

  • FOOTER:

    Es el pie de página del proyecto, aquí va la información de contacto o una firma.
